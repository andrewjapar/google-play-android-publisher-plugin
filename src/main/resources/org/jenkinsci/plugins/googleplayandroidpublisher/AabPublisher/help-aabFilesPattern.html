Specifies filenames or patterns matching one or more AABs that should be uploaded to Google Play.

You can use wildcards like "**/build/outputs/*/*-release.aab".
See the 'includes' attribute of Ant's FileSet for the exact format.
Note that multiple entries must be comma-separated.

The base directory is the build's workspace. You can only upload AABs that are located in your workspace.

This field supports substituting environment variables in the form ${SOME_VARIABLE} or $SOME_VARIABLE at build time.
